# Web Module

**Covers:** frontend work in Framer, React, Tailwind. Component design, design-system rules, accessibility.

**Activate when:** the project's `.claude-modules` file lists `web`. Loads `MODULE.md` + `rules/*.md` on session start. Skills and agents on demand.

**Status:** skeleton — expand with concrete preferences as they emerge.

## Table of Contents

### Rules (loaded with module)
- `rules/framer-conventions.md` — Framer Code Component structure, prop typing, responsive variants
- `rules/component-structure.md` — modular files, composition, no mega-files, props typed
- `rules/design-system.md` — no purple-hue default, no emojis in code, taste rules

### Skills (on demand)
- `skills/tailwind-patterns.md` — common Tailwind compositions
- `skills/framer-component-template.md` — boilerplate Framer Code Component

### Agents (on demand)
- `agents/component-architect.md` — plans component tree before code
- `agents/design-reviewer.md` — reviews for visual quality + accessibility

### Commands (on demand)
- `commands/new-component.md` — spec-first component creation
